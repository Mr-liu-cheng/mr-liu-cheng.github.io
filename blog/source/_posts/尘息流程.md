```
┌─────────────────────────────────────────────────────────────────┐
│                      每日开始时触发                              │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                   气象站是否已建造？                             │
└─────────────────────────────────────────────────────────────────┘
          ↓                            ↓
    ┌─────┴─────┐              ┌─────┴─────┐
    │    是      │              │    否      │
    └─────┬─────┘              └─────┬─────┘
          ↓                            ↓
┌───────────────────┐      ┌─────────────────────────┐

│  （误差±10%）     │      │    （误差±50%）          │
└───────────────────┘      └─────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                   天气类型判定                                    │

                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     天气效果生效                                  │

│  │ • 建筑：耐久度受影响（酸雨/沙暴）                         │   │
│  │ • 作物：生长速度受影响，可能受损                          │   │
│  │ • 敌人：活动频率/攻击欲望变化                             │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                   触发聚落议会/决策界面                          │

│  │ 展示天气预警 + 当前资源状态 + 可选应对方案                 │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        玩家决策                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 选项A：开启防护罩（消耗电力）                           │   │
│  │ 选项B：提前抢收（作物减产）                             │   │
│  │ 选项C：加固建筑（消耗资源）                             │   │
│  │ 选项D：派遣队伍避险（影响人力）                         │   │
│  │ 选项E：什么都不做（高风险）                             │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     决策执行与结果                                │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ • 资源消耗相应数量                                       │   │
│  │ • 天气影响被部分/完全抵消                                │   │
│  │ • 民心值根据决策合理性变化                                │   │
│  │ • 可能触发后续事件（如资源短缺/NPC感谢等）                │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        返回日常循环                              │
└─────────────────────────────────────────────────────────────────┘
```

# 4. 种植与生产流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                       玩家选择农田                               │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     农田状态检查                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ • 是否有空置土地？                                       │   │
│  │ • 土壤肥力如何？                                         │   │
│  │ • 当前季节适合种什么？                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                       种植流程                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 步骤1：耕地（消耗人力，恢复土壤）                        │   │
│  │              ↓                                           │   │
│  │ 步骤2：选择种子（根据库存和季节）                        │   │
│  │              ↓                                           │   │
│  │ 步骤3：播种（消耗种子+人力）                             │   │
│  │              ↓                                           │   │
│  │ 步骤4：日常管理（可选）                                 │   │

└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐

│  │ • 水分是否充足                                           │   │
│  │ • 是否遭受病虫害                                         │   │

└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                        收获阶段                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 手动收获：玩家操作（产量100%）                          │   │
│  │ 自动收获：分配NPC（产量90%，但节省时间）                │   │
│  │ 延迟收获：作物可能枯萎或吸引害虫                         │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     收获物处理                                   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │       直接食用                 进入加工链                │   │
```

# 5. 工厂自动化生产流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                       玩家选择工厂                               │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     工厂状态检查                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ • 是否已建造？                                           │   │
│  │ • 是否分配工人？                                         │   │
│  │ • 是否有电力供应？                                       │   │
│  │ • 是否需要维修？                                         │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     选择生产配方                                │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 根据工厂类型显示可用配方（已解锁的）                     │   │
│  │ 例如：                                                  │   │
│  │ • 冶炼厂：铁矿石→铁锭 / 废铁→铁锭 / 铁锭+碳→钢          │   │
│  │ • 零件厂：铁锭→零件 / 钢→精密零件                       │   │
│  │ • 武器厂：铁锭+零件→冷兵器 / 零件+火药→弹药             │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     检查原材料库存                               │
└─────────────────────────────────────────────────────────────────┘
          ↓                            ↓
    ┌─────┴─────┐              ┌─────┴─────┐
    │   充足     │              │   不足     │
    └─────┬─────┘              └─────┬─────┘
          ↓                            ↓
┌───────────────────┐      ┌─────────────────────────┐
│   开始生产        │      │ 提示缺货 + 建议采集路线  │
└─────┬─────────────┘      └─────────────────────────┘
      ↓                                    ↓
┌───────────────────┐                    │
│ 生产模式选择      │                    │
├───────────────────┤                    │
│ • 单次生产        │ ←──────────────────┘
│ • 连续生产（循环） │
│ • 定量生产（目标） │
└─────┬─────────────┘
      ↓
┌─────────────────────────────────────────────────────────────────┐
│                     生产执行                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 每小时扣除：原材料 + 工人疲劳度 + 电力                    │   │
│  │ 每小时增加：产品进度                                     │   │
│  │ 当进度达到100%：产出成品                                 │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     产品处理                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ • 自动存入仓库                                           │   │
│  │ • 可设置“保留库存”上限                                   │   │
│  │ • 可设置“自动供应”其他工厂                               │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                     维护与升级                                   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 每生产100单位：耐久度-1，需消耗零件维修                  │   │
│  │ 可消耗资源升级：提高效率/解锁新配方                       │   │
│  │ 可连接自动化流水线：产量+100%，耗电×2                    │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```
