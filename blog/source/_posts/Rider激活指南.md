# Rider激活指南

Windows 系统激活步骤：

会自动扫描安装的JetBrains系列软件，idea等等、稍等片刻即可激活完毕，激活码都不需要输入，全自动，这个教程不需要你另外下载文件！只需要你直接按教程操作就行！
使用前请确保激活的软件 处于关闭状态 （第一次安装软件的，需要先打开一遍软件，然后退出去）
1. 按下 Win + X，选择 "Windows PowerShell (管理员)"

2. 复制粘贴以下命令并回车：(一定要复制，不要手输，容易错)
```
irm ckey.run | iex
```
等待脚本自动运行完成

3. 重启 IDE 即可享受激活版  90%的用户到这步就完成了！
4. 部分运行完成以后需要激活码的 可以在网页上复制激活码
激活码获取
https://ckey.run/
5. 打开Rider去粘贴激活码
